Smart business payment solutions that allow your business to pay and get paid on time.

== Description ==
Enable Unifiedpost payment gateway in your WooCommerce store. Automate your payment processes so that your business pays and gets paid on time.

== Installation ==
* Install and activate the plugin
* Navigate to WooCommerce -> Settings -> Payments -> Unifiedpost
* Enable the gateway, save the API key in the settings
